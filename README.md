# ひといき

呼吸に意識を向けるためのガイドです。

電車の中で、会議の前に、眠れない夜に。
画面をタップすると円がゆっくり動きはじめるので、それに合わせて呼吸するだけ。むずかしい操作や登録はいりません。

**▶ 使ってみる: https://oekazuma.github.io/hitoiki/**

## 使い方

1. 画面のどこでもいいので、タップします
2. 円のうごきに合わせて「すって」「とめて」「はいて」
3. 終わりたくなったら、もう一度タップ

それだけです。

## 呼吸のパターン

はじめから3つのパターンを用意しています。「せってい」から切り替えたり、秒数を自分好みに調整したりできます。

| パターン               | すう | とめる | はく | とめる |
| ---------------------- | ---- | ------ | ---- | ------ |
| ゆっくり(はじめの設定) | 4秒  | –      | 6秒  | –      |
| 4-7-8呼吸              | 4秒  | 7秒    | 8秒  | –      |
| ボックス呼吸           | 4秒  | 4秒    | 4秒  | 4秒    |

## テーマ

時間帯に合わせて、朝は「うみ」、昼は「もり」、夕方は「ゆうやけ」、夜は「よぞら」へと配色がゆるやかに移ろいます。
「つきあかり」「くも」を含む6つのテーマから、好きなものを選んで固定することもできます。どれも落ち着いた、目にやさしい色合いです。

## どこでも使える

- **オフライン対応**: 一度開けば、地下鉄や飛行機の中など電波のない場所でも動きます
- **ホーム画面に追加**: スマートフォンのブラウザから「ホーム画面に追加」すると、アプリのようにすぐ開けます
- **音は出ません**: 静かな場所でも周りを気にせず使えます
- **画面は消えません**: ガイドの途中で画面がスリープしないよう配慮しています

## アクセシビリティ

- 呼吸のフェーズを 円の動き・ことば・進捗リング の複数の手がかりで伝えるため、色の区別に頼らず使えます
- すべてのテーマで、文字と背景のコントラスト(WCAG 2.1 AA)を確保しています
- スクリーンリーダーによるフェーズの読み上げに対応しています
- キーボードだけでも、すべての操作ができます
- OS の「視差効果を減らす」設定をオンにすると、円の動きがより控えめになります
