# ひといき

呼吸に意識を向けるためのガイドです。
画面のガイドに合わせて呼吸するだけで使えます。

## 特徴

- **考えさせない**: 開いて「はじめる」を押すだけ。秒数の判断は不要
- **オフラインで動く**: PWA として全アセットをキャッシュ。地下鉄や機内でも起動できます
- **音を出さない**: 公共交通機関でも周囲を気にせず使えます
- **色に頼らない**: フェーズは 形(円の拡大縮小)・テキスト・進捗リング の複数の手がかりで伝えます

## プリセット

| プリセット           | すう | とめる | はく | とめる |
| -------------------- | ---- | ------ | ---- | ------ |
| ゆっくり(デフォルト) | 4秒  | -      | 6秒  | -      |
| 4-7-8呼吸            | 4秒  | 7秒    | 8秒  | -      |
| ボックス呼吸         | 4秒  | 4秒    | 4秒  | 4秒    |

各フェーズの秒数は設定から自由に調整できます。

## 開発

```bash
pnpm install
pnpm dev        # 開発サーバー
pnpm test       # テスト(watch)
pnpm test:run   # テスト(1回)
pnpm build      # 本番ビルド
pnpm preview    # ビルドのプレビュー
pnpm check      # 型チェック
pnpm icons      # アイコンPNGの再生成
```
